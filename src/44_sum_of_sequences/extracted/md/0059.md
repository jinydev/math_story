“지난 시간에 선생님이 1부터 100까지 SS 구했을 때, 1부터
100까지 차례대로 쓴 자연수도 등차수열인 거죠?"

1, 2, 3, 4, …, 99, 100이라는 숫자를 보면 앞의 수에 1을 더해
서 뒤의 수가 되죠? 이 수열은 제1항이 1이고 공차가 1인 수열

로, 제100항까지 나열한 것이람니다.

1, 2, 3, 4, …, 99, 100이 등차수열이라는 것까지 기억하다니
정말 기억력이 대단한걸요. 그럼 1부터 100까지 합을 구하는 방
법도 기억하고 있겠군요. 그럼 1, 5, 9, 13, 17의 모든 항의 합은
어떻게 될까요?

“정말 간단한데요? 1, 5, 9, 13, 17을 거꾸로 하여 쓰면 합이
18이므로, 합이 18인 사각형이 5개가 되잖아요. 그럼 이 합을 2
로 나누면 되죠.”

18×5 - 45
