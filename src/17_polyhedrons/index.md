# 12. 다면체 (Polyhedron: 우주의 뼈대 설계도)

## 이 단원의 핵심 (Chapter Focus)
종이에 그려진 2차원 다각형들이 기지개를 켜고 z축을 향해 돌출하며 만들어 낸 각진 상자, **다면체(Polyhedron)**의 미학을 파헤칩니다. 
현대 컴퓨터 그래픽 데이터 매핑의 기초가 되는 2D 점선 '단면 전개도' 파이썬 코딩을 시작으로 고대 학자들이 세상을 지배하는 5개 원소로 지칭했던 절대 대칭 신의 조각 '플라톤 입체(정다면체)'의 3D SVG 시각화 모델을 관찰합니다. 
더 나아가 왜 정다면체가 5가지뿐인지를 파이썬 각도 검증기로 철퇴를 내리고, 구멍 뚫리지 않은 모든 상자를 관통하는 $V-E+F=2$ 오일러의 점선면 기하학 공식 검출 로직 작성, 그리고 정다각형의 뿔을 잘라내어 탄생한 월드컵 '축구공' 아르키메데스 절단 공법까지 3차원 폴리곤 공간의 마력을 100% 흡수하세요!
