# 수학이야기 52. 정규분포 (Normal Distribution)

## 📌 학습 개요
딱딱하게 끊어지는 동전과 주사위의 1차원 도박장(이산)에서 벗어나, 시간과 몸무게처럼 나노 단위로 쪼개지는 부드러운 우주의 곡선 **'연속확률분포(Continuous Distribution)'** 의 광활한 데이터 세계로 잠수합니다. "어떤 수가 정확히 나올 확률은 0퍼센트다"라는 파괴적 철학을 시작으로, 구간의 면적을 색칠해 내는 **확률밀도함수(PDF)** 의 적분 룰을 세팅합니다. 최종적으로 무질서한 세계가 스스로 조립되어 만들어내는 우주 최강의 대칭 신전 **'정규분포(Bell Curve)'** 의 탄생과, 이 얽히고설킨 모든 혼란을 평균 0, 편차 1 단위의 백분율로 한 방에 줄 세워버리는 무자비한 환산 스캐너 **'Z 표준화 기술'** 을 파이썬 코드로 체화합니다.
