# 수학이야기 82. 이차부등식 (Quadratic Inequalities)

## 📌 학습 개요
방정식이 "저울이 완벽하게 0 과 맞아떨어지는 바늘구멍 같은 정답 점(Point)" 을 찾는 모범생들의 수학이라면, 부등식(Inequality) 은 엔진이 폭발하기 전까지 아슬아슬하게 견뎌야 하는 압력 임계치나, 무거운 폭약이 실어주는 편향된 **어마어마한 허용 구역(Area)** 을 찾아내는 서바이벌 기하학입니다.
이번 82장에서는 이차함수의 포물선을 이용해 수면(0) 위로 날아가는지 바다 밑으로 가라앉는지를 눈으로 스캐닝하는 **그래프 해킹 기법**, 어떤 우주 쓰레기가 부딪혀도 평생 거뜬히 공중부양할 수 있도록 근을 아예 말살시켜 버리는 **판별식($D<0$) 역발상 치트키**, 수문 1과 수문 2를 겹쳐 쳐서 비가 절대 새지 않는 이중 보안 벙커(AND 논리곱) 교집합을 도출해 내는 **연립이차부등식**, 마지막으로 로봇청소기의 장애물 사거리를 탐지하는 원점 거리 추적기인 **절댓값 $|x|$** 의 기하학적 찢기 원리까지 파이썬의 배열 마스킹과 빅데이터 필터링 관점으로 완전히 해부합니다.

## 📚 목차 (Table of Contents)

1. [1. 점(Point) 에서 구역(Area) 으로: '이차부등식의 탄생'](01_concept)
   - 부등식이란 딱 떨어지는 정답 1개가 아니라 살아남는 무한한 우주의 영토이며, 방정식의 '근' 은 그 영토를 에워싸는 감옥의 철조망(경계선) 이 됨을 확인합니다.
2. [2. X축 수면 위로 솟아오른 자: '그래프와 부등식'](02_graphic_solution)
   - 복잡한 덧뺄셈 대수식을 버리고 포물선 함수 $y=f(x)$ 2D 그림 하나 던져 놓은 뒤, 땅 위(양수) 와 물 밑(음수) 을 눈으로 판정하는 사기적인 시야를 탑재합니다.
3. [3. 땅에 닿지 않는 영원한 비행: '판별식과 절대부등식'](03_discriminant)
   - "모든 $x$에 대해 무조건 0보다 커야 해!" 라는 무적 실드를 켜기 위해선, 이차방정식의 바닥 충돌점(근) 이 아예 없어야 함($D<0$) 을 깨닫습니다.
4. [4. 방패와 창의 교차 구역: '연립이차부등식'](04_system)
   - 수십 개의 수직선을 레이어처럼 겹겹이 스택(Stack) 시킨 후, 이 모든 우산을 뚫어내어 살아남은 유일한 공통 구역(교집합) 만을 스캐닝하는 AND(논리곱) 보안 필터입니다.
5. [5. 거리 감각을 마비시키는 껍질: '절댓값과 부등식'](05_absolute)
   - 절댓값 공포증을 부수고 $|$ 기호를 "나와 타겟 지점 사이의 도보 물리 거리(Distance)" 로 치환하는 마인드 세스 전환과, 자율주행 레이더 충돌 알고리즘에 적용합니다.
