# 수학이야기 48. 일차부등식 (Linear Inequalities)

## 📌 학습 개요
오직 정답이 평면상의 점 1개로 확정 나버리는 갑갑한 '방정식($=$)'의 감옥에서 뛰쳐나와, 영역과 구간이라는 기하학적 평원을 달리는 **'부등식($>, \le$)'** 세계의 웅장함을 마스터합니다.
더하거나 뺄 때는 끄떡없던 부등호의 아가리가 음수($-$) 곱셈 한 방에 미친 듯이 까뒤집히는 수학계 최악의 함정을 파훼하며, 파이썬(Python)의 비교 데이터 연산자(Boolean: `True/False`)로 일상생활의 예산 삭감과 생존 한계선을 자율주행 알고리즘처럼 해킹하는 스킬을 배웁니다.
