# 02. 두 번째 수업: 코드로 우주를 그리다, 파이썬 재귀 함수 (Recursion)

프랙탈의 핵심은 **"자신의 복사본을 자기 한가운데에 계속 집어넣는 것"**입니다.
컴퓨터 공학에서는 이처럼 '자기 자신이 자기 자신을 계속해서 부르는 현상'을 **재귀(Recursion)**라고 부릅니다. 현대 프로그래밍 언어의 궁극기이자 수많은 프로그래머들을 멘붕에 빠지게 하는 가장 난해한 개념이기도 하죠.

---

## 학습 목표
* "함수가 함수 자신을 다시 호출한다"는 프로그래밍의 **재귀(Recursion)** 패턴을 이해합니다.
* 무한 루프에 빠지지 않기 위한 재귀 함수의 필수 조건, **베이스 케이스(Base Case)**를 파악합니다.
* 프랙탈 그리기나 디렉터리 검색 등, 파이썬 코드에서 재귀가 어떻게 실전으로 운용되는지 경험합니다.

## 1. "사과 안에 사과 안에 사과..."

거울 두 개를 마주 보게 두고 그사이에 서 본 적이 있나요?
거울 속의 내 거울 속에, 다시 거울 속의 내 거울 속이 끝없이 이어지며 무한한 심연이 펼쳐집니다. 이것이 시각적인 '재귀'입니다.

파이썬 코드에서 이를 구현하려면 어떻게 해야 할까요? 
함수(`def`) 하나를 만들고, 그 함수 안에서 일을 시킬 때 방금 만든 자기 자신의 함수 이름을 또 불러버리면 됩니다. 

## 2. 파이썬 재귀 함수를 위한 안전벨트 (Base Case)

아무 생각 없이 자기 자신을 호출하는 함수를 만들면, 컴퓨터 메모리가 터져 나갈 때까지(Stack Overflow Error) 영원히 우주를 향해 파고들다가 프로그램이 강제 종료됩니다.

그래서 재귀 함수를 짤 때는 무조건 **"언제 그만둘 것인가? (탈출 조건)"**를 제일 먼저 코드 첫 줄에 박아 두어야 합니다. 이를 **베이스 케이스(Base Case)**라고 부릅니다. 
우리가 흔히 구하는 팩토리얼($5! = 5 \times 4 \times 3 \times 2 \times 1$) 수식 계산을 파이썬 재귀로 짜봅시다.

```python
# 파이썬 재귀(Recursion)로 구현한 팩토리얼 폭탄!

def calc_factorial(n):
    
    # 1. 안전벨트 (Base Case): 
    # 숫자가 깎여서 1까지 도달하면, 제발 그만 파고들고 1을 반환하며 탈출해라!
    if n == 1:
        return 1
        
    # 2. 자기 원본 함수 다시 부르기 (Recursion):
    # 나 자신(n)과, "나보다 1 작은 폭탄"을 계산하러 떠난 나의 복제본 함수를 곱한다!
    return n * calc_factorial(n - 1)

# 메인 실행 엔진
result = calc_factorial(5)
print(f"5! 의 재귀 연산 결과: {result}")
# 출력: 5! 의 재귀 연산 결과: 120
```

`calc_factorial(5)` 가 불리면, 얘는 지가 일을 끝내지 않고 `calc_factorial(4)` 를 호출합니다. `(4)`는 또 `(3)`을, `(3)`은 `(2)`를 부르면서 시한폭탄을 계속 옆 사람에게 넘깁니다. 
그러다가 마지막 안전벨트 인 `n == 1` 을 만나는 순간 폭탄 넘기기가 중단되고, $1 \times 2 \times 3 \times 4 \times 5$ 가 연쇄적으로 탁탁탁 터지며 120이라는 결과를 역순으로 뱉어내는 놀라운 예술입니다.

## 3. 프랙탈 폴리곤 렌더링의 심장

아까 배웠던 코흐의 눈송이 프랙탈을 컴퓨터 그래픽으로 렌더링하려면 반복문(`for`) 따위로는 죽었다 깨어나도 못 그립니다. 단계가 올라갈수록 선의 위치가 사방으로 흩어지기 때문입니다.
하지만 재귀 함수(`draw_koch()`)를 이용하면 놀랍도록 싱겁게 해결됩니다. 

> "선을 3등분으로 쪼개고, 그 작은 선분들 하나하나에 대해서 다시 **나를 호출(`draw_koch()`)**해라!" 

이 한 줄의 마법 주문 덕분에, 프로그래머는 단 10줄의 코드로 수천만 개의 폴리곤 선이 화면을 자글자글하게 뒤덮는 복잡한 프랙탈 우주를 창조할 수 있습니다. 

## 학습 정리
1. **재귀(Recursion)**: 컴퓨터 함수가 로직 처리를 위해 자기 자신을 또 호출하여 거울 속의 거울처럼 깊이 파고드는 프로그래밍 아키텍처. 프랙탈 기하학이 코드로 환생한 모습이다.
2. **베이스 케이스 (Base Case)**: 끝없는 무한 루프 에러(스택 오버플로우)를 막기 위해, 일정 깊이에 도달하면 함수가 파고들기를 멈추고 거꾸로 값을 반환하며 되돌아오게 만드는 탈출 제어문.
3. 코흐의 곡선이나 디렉터리 내 수만 개의 하위 폴더 검색처럼 "가지치기(Branching)"가 끝없이 이루어지는 자연계 구조는 오로지 파이썬의 이 재귀 코드로만 통제 가능하다.
