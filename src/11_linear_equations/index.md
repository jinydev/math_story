# 13. 일차방정식 (Linear Equation: 미지의 엑스(x)를 향한 추리)

## 이 단원의 핵심 (Chapter Focus)
손가락으로 셈을 하던 산술의 시대를 넘어, 보이지 않는 숫자의 정체를 기호로 묶고 해독하는 **'대수학(Algebra)'**의 혁명을 체험합니다. 
대수학의 아버지 '디오판토스'의 묘비 미스터리로 시작해, 변수 기호 '$x$'가 코딩 알고리즘에서 얼마나 강력하게 작용하는지 파악합니다. 수식을 양팔 저울(SVG)에 올려놓고 밸런스를 잡는 '등식의 성질'을 체득하며, 파이썬 기호 수학(`SymPy`) 봇을 구동하여 1차 방정식을 자동 풀이해 봅니다. 
또한 《구장산술》의 동양 산가지 AI 툰을 통해 연립방정식의 기원을 여행하고, "식을 푼다"라는 행위가 "그래프의 교차점"을 구하는 것과 완벽히 동치라는 위대한 기하학적 융합을 파이썬 `matplotlib` 2D 스캐너로 목격하게 될 것입니다!
