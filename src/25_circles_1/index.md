# 수학이야기 28.원1 (Circle 1: The Perfect Loop)

인류 역사상 인간의 삶을 가장 폭발적으로 진화시킨 발명품은 불도, 전기도, 컴퓨터도 아닙니다. 바로 **기원전 3500년 수메르인이 통나무를 썰어 만들어낸 굴러가는 원판, '수레바퀴(Wheel)'** 입니다. 

이 하나의 단순하고 완벽한 기하학적 곡선이 없었다면 마차도 굴러가지 않았고, 거대한 돌을 쌓는 도르래 시스템도 불가능했으며, 톱니바퀴 모터와 하드디스크가 회전하는 현대 문명 전체가 탄생하지 못했습니다. 도대체 완벽한 동그라미(원, Circle) 의 본질이 무엇이길래 이렇게 우주의 모든 에너지를 마찰 없이 효율적으로 순환시키는 것일까요?

이번 **원(Circle) 파트 1** 에서는 원의 가장 근본적인 해부학을 다룹니다. 중심과 반지름, 부채꼴의 넓이, 그리고 영원히 풀리지 않는 인류의 무한 스크립트 비밀번호인 **파이($\pi$, 원주율)** 의 정체를 파이썬 코드로 철저하게 분해하여 해석해 드리겠습니다.
