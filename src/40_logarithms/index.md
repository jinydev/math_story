# 수학이야기 39. 로그 (Logarithms)

## 📌 학습 개요
17세기, 평생 동안 곱셈 계산에 파묻혀 쓰러져 가던 천문학자들을 구원하기 위해 네이피어가 발명한 **'로그(Log)'**. 
어깨 위에 앉아있던 거듭제곱의 지수를 주인공 위치로 끌어내려 복잡한 계산을 초등학생 수준의 단순한 덧셈으로 해체해 버리는 과정과, 파이썬 프로그래밍에 활용되는 상용로그와 로그 스케일을 알아봅니다.
