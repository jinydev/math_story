# 10. 암호 (Cryptography: 비밀 수학의 세계)

## 이 단원의 핵심 (Chapter Focus)
가장 원초적인 문장 치환부터 현대 수백 억 달러의 자산이 오고 가는 최첨단 블록체인 보호 생태계까지, 창과 방패처럼 진화해 온 **'수학 암호 방어 체계'**를 파헤칩니다. 알파벳을 회전시키는 일차원적인 고대 모듈러(Modulo) 연산이 인간의 정교한 언어 통계 데이터를 분석하는 데이터 사이언스(matplotlib 차트 시뮬레이션) 앞에서는 얼마나 무참하게 부서지는지 체험합니다.
더 나아가 소인수분해의 극한 비대칭성을 활용해 인터넷 세상을 구원한 불멸의 'RSA 공개키 방정식', 파이썬 프로그래밍 실습, 비밀을 입 밖으로 꺼내지 않고도 내가 나임을 100% 증명하는 신기루 같은 영지식 증명(Zero-Knowledge Proof)까지. 해커들의 눈을 완벽하게 가리기 위한 천재들의 치열한 두뇌 코딩 게임을 감상하세요!
