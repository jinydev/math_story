# 수학이야기 37. 평면곡선 (Plane Curves)

## 📌 학습 개요
이 챕터에서는 무심코 지나치는 구불구불한 도로, 타원형 트랙, 은하의 나선구조가 가진 아름다움을 **'평면곡선'**의 시점에서 해석합니다. 단순히 $y=f(x)$에 갇혀 있지 않는 **극좌표**와 공학적 식을 **파이썬(Python)**의 시뮬레이션을 통해 직접 경험해 봅니다.
