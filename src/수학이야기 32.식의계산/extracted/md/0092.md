①번 색종이 넓이 $= 2x = x + x$
+) ②번 색종이 넓이 $= 3x = x + x + x$
------------------------------------
두 색종이의 합 $= 2x + 3x = x + x + x + x + x$

그래서 두 색종이의 넓이의 합은 넓이가 $x\text{cm}^2$인 ③번 색종이가 5개 들어가므로 $5x$입니다.

$$2x + 3x = x + x + x + x + x = 5x$$

이번에는 ②번 색종이에서 ①번 색종이의 넓이를 빼 봅시다.
