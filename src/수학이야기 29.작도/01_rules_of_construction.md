# 01. 절대 반지 규칙: 눈금 없는 자와 컴퍼스

## 1. 학습 목표 (Learning Objectives)
* 유클리드의 세계에서 기하학의 해커들이 도형을 복제할 때 사용하는 유일한 엔진 스펙, **'눈금 없는 자' 와 '컴퍼스' 의 물리적 제한값 조항** 들을 명확히 이해합니다.
* 뻔해 보이는 "선을 옮기기", "각을 이등분하기" 같은 작업이 이 시스템 제약 하에서 어떻게 컴퍼스의 궤도 아키텍처로 증명되는지 메커니즘을 파악합니다.

## 2. 하드코어 룰: 도구의 스펙 제한 모델
작도는 마치 무기만 들려주고 치트키 사용을 전부 밴(Ban) 당한 하드코어 액션 게임과 같습니다. 이 게임의 입력 기기는 단 2개뿐입니다.

1. **[직선 추출 봇]: 눈금 없는 자 (Straightedge)**
   * **허용 기능**: 좌표 평면의 아무 곳에 찍힌 임의의 "점 두 개"를 정확히 한 치의 오차도 없이 잇는 최단 거리 직선을 쭉 발사하는 기능.
   * **금지 기능**: "엇? 이 선의 길이가 어느 정도 되나 봐야지~" 하고 눈금을 이용해 절대적인 크기를 계측하는 행위는 **시스템 반칙** 으로 실격(Fail) 처리됩니다. 그냥 긋기만 할 뿐입니다.

2. **[반경 복제 스텔스]: 컴퍼스 (Compass)**
   * **허용 기능**: 맵의 아무 점에나 뾰족한 침(중심)을 박고, 어떤 점까지 연필을 벌린 채로 그대로 한 바퀴를 돌려 "완벽한 타깃의 복제 궤도(원)"를 남기는 기능.
   * **금지 기능**: "이 컴퍼스 벌어진 각도 그대로 공중에 들고 저~쪽 위치로 날아가서 갖다 대 놔서 복사 해봐야지!" 안 됩니다. 고대의 룰은 **컴퍼스를 종이에서 떼는 순간 나사가 헐거워져 벌어진 폭(반지름)이 초기화(0)되어버린다** 고 가정하는 미친 페널티를 부여했습니다.
   * (따라서 길이를 옮기려면 다른 점들을 매개체 삼아 서로 겹치는 여러 개의 원을 무식하게 계속 그려가며 릴레이 복사를 치는 고도의 프로토콜이 필요합니다. "무너지는 컴퍼스 제약" - 다행히 후대 수학자들이 이 제약과 "안 무너지는 컴퍼스"가 논리적으로 동등함을 증명하여, 요즘엔 폭을 유지한 채 떼서 옮겨 그리는 룰을 허용합니다.)

## 3. 작도 스킬 1: 선분의 수직이등분선 조준!
책상에 덩그러니 선분 $AB$ 가 놓여 있습니다. 자에 눈금이 없으니 자를 대서 "음 여기가 딱 $5 cm$ 네, 절반 뚝 베자!" 하는 얌체짓을 할 수 없습니다. 이 선분을 수직 방향으로 완벽하게 반갈죽(이등분) 내버리려면 어떻게 할까요?

1. 선분 왼쪽 끝 점 $A$ 에 컴퍼스의 뾰족 침을 박습니다. 길이는 절반보다 살짝 더 길게 대충 벌려 놓고 허공 위아래로 슥~ 궤적(호)을 그립니다.
2. 컴퍼스 폭을 **절대로 바꾸지 않은 채**, 오른쪽 점 $B$ 로 달려가 뾰족 침을 스왑 박습니다. 그리고 1번과 똑같이 위아래로 슥~ 궤적(호)을 그립니다.
3. 이 두 개의 위도, 경도 궤적이 서로 교차하며 하늘에 **$\times$ 자 크로스 포인트(교점)** 가 위아래로 딱 2개 잡힙니다.
4. 이제 눈금 없는 자를 대고 위아래 교차점 두 개를 잇는 궤도를 관통시킵니다.
5. 챙! 선분 $AB$ 는 완벽한 각도 $90^\circ$ 도의 레이저를 맞고 반토막 나버렸습니다. 오차 $0\%$.

## 4. 작도 스킬 2: 각의 이등분선 레이저!
날카롭게 예각이 뿜어 나와 V자로 벌어진 각 $O$(Origin)가 있습니다. 이 각의 빈 공간을 오차 없이 반으로 가르는 레이저 선은 어떻게 도출할까요?

1. 무심하게 꼭짓점 $O$ 에 컴퍼스를 박고 아무 각도로나 휙 가로질러 날리는 원호를 하나 그어 $V$자 선분 두 개를 긁어버립니다. (교차점 $X, Y$ 생성)
2. 교점 $X$ 에 컴퍼스를 박고 허공에 원호를 훅 날립니다. 폭을 고정합니다.
3. 교점 $Y$ 로 이동해 침을 박고 다시 원호를 훅 날립니다.
4. 2번과 3번의 궤적이 겹치면서 생성된 교점 타깃 포인트 $Z$!
5. 꼭짓점 $O$ 와 최종 교점 타깃 $Z$ 를 눈금 없는 자로 관통해 버리면?
6. 퍼펙트! 각이 완벽하게 둘로 쪼개졌습니다. 이것이 자와 컴퍼스만으로 증명된 기하학적 마법입니다.

## 5. 학습 정리 (Summary)
1. **눈금 없는 자의 본질**: 이 우주에 한 쌍의 두 점을 최단 거리(직선)로 잇는 통로를 개통시켜 주는 유일한 커넥터(Connector)입니다. 길이는 절대 잴 수 없습니다.
2. **컴퍼스의 본질**: 공간 상에 존재하는 특정 "길이/거리 스탯값"을 각도/방향에 상관없이 모든 평면에 복사 $\cdot$ 붙여넣기(Copy $\&$ Paste) 해주는 클로닝(Cloning) 기능 모듈 도구입니다.
