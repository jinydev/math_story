# 03. 세 번째 수업: 코드로 찾는 삼각형의 무게중심 (Centers of a Triangle)

세상에 존재하는 모든 물체는 떨어지지 않고 완벽한 균형을 이룰 수 있는 단 하나의 마법 같은 점을 가지고 있습니다. 이를 **무게중심(Center of Gravity)**이라고 부릅니다. 
우주에서 가장 안정적인 도형인 삼각형 또한 그 내부에 완벽한 균형점인 무게중심($G$)을 품고 있습니다.

---

## 학습 목표
* 삼각형의 세 중선이 만나는 무게중심($G$)의 기하학적 의미와 2:1의 황금 비율을 이해합니다.
* 외심(Circumcenter)과 내심(Incenter)의 차이점을 가볍게 짚고 넘어갑니다.
* 파이썬(Python) 좌표 리스트를 이용해 3D 그래픽 물리 엔진이 삼각형의 무게중심을 구하는 공식을 직접 구현해 봅니다.

## 1. 완벽한 균형점, 무게중심 ($G$)

삼각형의 한 꼭짓점에서 마주 보는 변의 한가운데(중점)로 선을 죽 그어봅시다. 이 선을 **중선(Median)**이라고 합니다.
삼각형에는 세 개의 꼭짓점이 있으니 중선도 세 개를 그릴 수 있는데, 소름 돋게도 이 세 개의 선은 반드시 삼각형 내부의 **단 하나의 점**에서 완벽하게 교차합니다. 이 점이 바로 삼각형의 **무게중심($G$, Gravity center)**입니다.

판지로 만든 삼각형의 이 $G$ 점에 샤프심을 콕 꽂아 올려두면, 삼각형은 어느 쪽으로도 기울지 않고 완벽한 수평을 유지합니다. 
무게중심을 지나는 중선에는 한 가지 놀라운 황금 비율이 숨어있는데, 꼭짓점에서 무게중심까지의 거리와, 무게중심에서 밑변까지의 거리가 항상 **$2:1$** 의 완벽한 비율을 이룬다는 것입니다. 

### (참고) 다른 중심들
- **외심(Circumcenter)**: 세 꼭짓점을 모두 지나는 바깥쪽 훌라후프(외접원)의 중심.
- **내심(Incenter)**: 세 변에 모두 닿는 안쪽 타이어(내접원)의 중심.

## 2. 물리 엔진의 코어 함수: 무게중심 좌표 구하기

컴퓨터 게임에서 날아가는 화살이나 굴러가는 바위의 물리 법칙(Gravity Physics)을 계산하기 위해, 컴퓨터는 초당 수백 번씩 물체의 무게중심 좌표$(x, y)$를 찾아냅니다.

삼각형의 세 꼭짓점 좌표만 안다면, 무게중심의 $x$좌표와 $y$좌표를 구하는 공식은 이 세상에서 가장 아름답고 단순합니다. 바로 **"세 좌표를 다 더한 뒤 3으로 나누는 것(평균)"**입니다.

> $G(x, y) = (\frac{x_1 + x_2 + x_3}{3}, \frac{y_1 + y_2 + y_3}{3})$

이를 파이썬 코드로 구현해 봅시다.

```python
# 파이썬으로 2D 삼각형의 무게중심 구하기

def get_centroid(p1, p2, p3):
    # 각 점의 x좌표들을 모아서 평균(3으로 나눔)을 낸다.
    center_x = (p1[0] + p2[0] + p3[0]) / 3
    # 각 점의 y좌표들을 모아서 평균을 낸다.
    center_y = (p1[1] + p2[1] + p3[1]) / 3
    
    return (center_x, center_y)

# 삼각형 세 꼭짓점 좌표 (x, y)
point_A = (1, 2)
point_B = (5, 3)
point_C = (3, 7)

# 마우스 클릭 한 번으로 무게중심 계산!
gravity_center = get_centroid(point_A, point_B, point_C)

print(f"세 꼭짓점: A{point_A}, B{point_B}, C{point_C}")
print(f"완벽한 균형을 이루는 무게중심 G의 좌표: {gravity_center}")

# 출력: 완벽한 균형을 이루는 무게중심 G의 좌표: (3.0, 4.0)
```

어떤 치열한 액션 게임이든, 총알이 날아와 폴리곤(삼각형) 몬스터를 타격하는 순간, 그래픽 엔진 내부에서는 위와 똑같은 코드가 작동하여 몬스터의 중심축이 무너지는 애니메이션을 계산해 냅니다. 수학이 곧 물리 엔진의 심장인 것입니다.

## 학습 정리
1. **무게중심($G$)**: 삼각형의 세 중선이 만나는 단 하나의 점으로, 물체가 완벽한 수평 균형을 이루는 지점이다.
2. **2:1 비율**: 무게중심은 꼭짓점으로부터 밑변 쪽으로 항상 거리의 3분의 2 지점($2:1$ 비율)에 위치한다.
3. **좌표 평균의 마법**: 2D 또는 3D 그래픽스에서 삼각형의 무게중심 좌표는 단순히 세 꼭짓점 좌표의 $x, y$ (산술 평균)을 내는 파이썬 코드 한 줄로 완벽히 일치하게 구해낼 수 있다.
