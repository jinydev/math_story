# 02. 두 번째 수업: 스케일링의 예술, 삼각형의 닮음 (Similarity)

만약 스마트폰에 뜬 엄마의 사진을 두 손가락으로 쫙 펼쳐서 확대(Zoom In)하면 어떨까요?
사진 속 엄마의 크기는 2배 3배로 커지겠지만, 코피가 날 정도로 얼굴의 비율이 깨지거나 각도가 찌그러지진 않습니다. 

이렇게 모양(비율과 각도)은 완벽하게 놔둔 채, 오직 크기만 확대하거나 축소한 관계를 **닮음(Similarity)**이라고 합니다.

---

## 학습 목표
* 실생활과 사진 비율에서 볼 수 있는 닮음표현 기호($\sim$)와 닮음비(Scale Factor)를 배웁니다.
* 합동 조건보다 훨씬 더 너그러운 AA 닮음(두 각도만 같음)에 대해 파악합니다.
* 파이썬과 행렬 수학에서 '스케일 변환(Scale Transform)'을 하는 원리를 깨우칩니다.

## 1. 뼈대 각도만 똑같다면 크기는 얼마든지

합동은 '도장 찍기' 였다면 닮음은 '고무줄 늘리기'입니다. 
닮은 두 도형을 기호로 쓸 때 수학자들은 영어 단어 Similar(비슷한)의 **$S$** 자를 옆으로 뉘어서 예쁜 물결 모양(**$\sim$**) 기호로 표시합니다.
> $\triangle ABC \sim \triangle DEF$ (삼각형 ABC와 DEF는 서로 닮은꼴이다)

닮음에서도 서로를 인정하는 최소 요건이 있습니다. 합동 조건과 묘하게 비슷합니다!
1. **SSS 닮음**: 세 짝궁 변의 '길이의 비율'이 일정하다! ($10cm:20cm$, $15cm:30cm$, $20cm:40cm$ 처럼 모두 두 배로 뻥튀기되었다면 모양은 완벽하게 똑같습니다.)
2. **SAS 닮음**: 두 변의 뻥튀기 비율이 같고, 그 **끼인 각**이 완벽히 똑같다! (벌어진 각도가 고정된 채 양팔 길이만 똑같이 길어지면 닮은꼴이 됩니다.)
3. **AA 닮음 (슈퍼 패스)**: 양 끝의 두 각도만 같으면 됩니다! 삼각형의 세 각의 합은 어차피 무조건 $180^\circ$ 이므로, 두 각만 같으면 남은 한 각도 저절로 같아집니다. 각도 세 개가 같다는 건 **뼈대의 기울기**가 완전히 똑같다는 뜻이기 때문에, 크기는 다를지언정 모양은 100% 똑같은 쌍둥이 삼각형이 됩니다.

## 2. 탈레스가 피라미드를 잰 비법 (AA 닮음)

도입부에서 보았던 탈레스의 지팡이 이야기를 떠올려 볼까요?
태양빛은 지구에 평행하게 내리쬡니다.
1. 지팡이 그림자의 끝 노란빛 각도 = 피라미드 그림자의 끝 노란빛 각도 (동위각으로 일치)
2. 지팡이가 모래에 꽂힌 수직 각도 = 피라미드가 바닥과 이루는 각도 ($90^\circ$ 로 일치)

두 개의 각도가 우연히 똑같습니다. 빙고! 바로 **AA 닮음**입니다.
작은 지팡이와 그림자가 1:1 비율($1m : 1m$) 이었다면, 거대한 피라미드의 그림자가 $146m$일 때 높이도 무조건 1:1 비율인 $146m$ 가 될 수밖에 없는 우주적 논리입니다.

## 3. 파이썬 그래픽스의 핵심 기술 (Scale Matrix)

포토샵에서 사진을 늘리거나, 3D 게임에서 유저가 몬스터에게 줌인을 땡길 때 컴퓨터 안에서는 어떤 연산이 일어날까요?
바로 삼각형(폴리곤) 꼭짓점들의 X, Y 좌표값을 일정한 '닮음비(Scale Factor)' 통에 넣어 스칼라 곱셉을 수행합니다.

```python
# 파이썬으로 2D 삼각형 좌표를 닮음비 2배로 확대(Scale) 해보기

# 원래 삼각형의 꼭짓점 3개 좌표 (x, y)
triangle_origin = [(1, 1), (3, 1), (2, 4)]
scale_factor = 2  # 2배로 크기를 키워보자! (닮음비 1:2)

# 닮음(확대) 삼각형의 좌표를 담을 모임통
scaled_triangle = []

# 기존의 점들을 하나씩 꺼내어 두 배로 뻥튀기(곱셉) 한다.
for x, y in triangle_origin:
    new_x = x * scale_factor
    new_y = y * scale_factor
    scaled_triangle.append((new_x, new_y))

print("원본 좌표 (작은 삼각형):", triangle_origin)
print("확대된 좌표 (큰 삼각형):", scaled_triangle)

# 출력: 원본 좌표 (작은 삼각형): [(1, 1), (3, 1), (2, 4)]
# 출력: 확대된 좌표 (큰 삼각형): [(2, 2), (6, 2), (4, 8)]
```
모든 점의 $x$축 거리와 $y$축 거리를 똑같은 비율($2배$)로 늘렸기 때문에, 새로 태어난 큰 삼각형은 원래 삼각형과 각도가 전혀 찌그러지지 않고 완벽한 **SAS(또는 SSS) 닮음** 상태를 유지합니다. 스마트폰 화면을 두 번 두드려 확대를 할 때, 매끄러운 줌 인 애니메이션은 바로 이 코드처럼 닮음비 $Scale$ 값이 $1.1$, $1.2$, $1.5$로 눈보다 빠르게 반복 연산되는 것입니다.

## 학습 정리
1. **닮음($\sim$)의 의미**: 모양(각도와 형태)은 똑같은데 크기만 확대/축소된 상태. 
2. **AA 닮음 (절대 조건)**: 삼각형의 세 각 중에 딱 '두 각도' 정보만 같아도 두 물체의 모양은 완전히 똑같다는 것을 보장받는다. 피라미드 측정의 1등 공신이다.
3. **컴퓨터 그래픽 스케일링**: 확대/축소(Zoom) 처리를 위해 각 모서리의 $(x, y)$ 좌표에 숫자를 직접 곱해 배율(Scale Factor)을 늘려가는 모든 알고리즘은 수학의 닮음 공식에 기반한다.
