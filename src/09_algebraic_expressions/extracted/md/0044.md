> **중요 포인트**
> **문자를 사용하여 곱셈을 나타내는 방법**
> ① 수와 문자, 문자와 문자의 곱에서는 곱셈 기호 $\times$를 생략하고, 문자는 알파벳 순서대로 씁니다.
> ② 문자와 수의 곱에서는 수를 문자 앞에 씁니다.
> ③ 어떤 수에 1을 곱해도 그 곱은 그 수 자신이 됩니다.

이번에는 문자를 사용하여 나눗셈을 간단하게 나타내는 방법을 알아봅시다.
$3\text{m}$짜리 색 테이프를 네 사람이 똑같이 나누어 가지려고 할 때, 한 사람이 가지는 색 테이프의 길이는 얼마인지 구해 볼까요?

|     |     |     |             |
| --- | --- | --- | ----------- |
| 가  | 나  | 다  | 라          |
| 가  | 나  | 다  | 라          |
| 가  | 나  | 다  | 라          |
| $0$ |     |     | $1\text{m}$ |

가, 나, 다, 라 네 명은 각각 $1\text{m}$의 $\frac{1}{4}$씩 갖게 됩니다.
