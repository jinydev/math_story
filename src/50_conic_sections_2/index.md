# 수학이야기 56.이차곡선2 (Conic Sections 2)

이차곡선 1부(모듈 45) 에서 포물선을 통해 로켓과 통신 위성 안테나의 기본 원리를 배웠다면, 이번 2부에서는 우주 시뮬레이션의 진짜 주인공들인 **"타원(Ellipse)"** 과 우주를 찢고 탈출해버리는 **"쌍곡선(Hyperbola)"** 의 방정식 코드를 탐구합니다.

태양계의 행성들은 왜 완벽한 원이 아니라 살짝 찌그러진 타원 모양으로 돌고 있을까요? GPS 위성은 지구에서 나의 위치를 정확히 잡아내기 위해 왜 쌍곡선의 거리 차이 렌더링 시스템을 사용할까요? 

단순히 $x^2 + y^2 = 1$ 같은 수식을 외우는 것을 넘어, 두 개의 초점(Foci) 이 허공에서 어떻게 서로를 당기고 밀어내며 이 우아하고 거대한 곡선의 알(Egg) 과 모래시계를 빚어내는지, 기하학적 해킹의 끝판왕 단원으로 여러분을 안내합니다.
